# 画像ファイルの配置方法

## AIKA18号のキャラクター画像を配置してください

1. `public/images/` ディレクトリに `aika-character.png` という名前で画像ファイルを配置してください。

2. 画像の推奨サイズ:
   - 幅: 400px以上
   - 高さ: 400px以上
   - 形式: PNG（透明背景推奨）またはJPG

3. 画像が配置されていない場合、プレースホルダーとして🥊アイコンが表示されます。

## 画像ファイルのパス

画像ファイルは以下のパスに配置してください:
```
public/images/aika-character.png
```

Web上では `/images/aika-character.png` としてアクセスされます。
