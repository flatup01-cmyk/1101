# 動画解析用プロンプトテンプレート集

## Dify動画解析用プロンプト

### 基本プロンプト

```
この動画の格闘家の技術的特徴を詳細に分析してください。

以下の項目について、具体的に記述してください：

1. **スタンス**
   - 基本スタンスの特徴（オーソドックス、サウスポーなど）
   - スタンスの幅、重心の位置
   - スタンスの動き方、変化の仕方

2. **パンチの技術**
   - ストレートの軌道と角度
   - フックの軌道と角度
   - アッパーカットの軌道と角度
   - パンチのスピードと威力
   - パンチのコンビネーションの特徴
   - パンチを打つ際の体重移動

3. **キックの技術**
   - フロントキックのフォームと角度
   - ミドルキックのフォームと角度
   - ハイキックのフォームと角度
   - ローキックのフォームと角度
   - キックの軌道とスピード
   - キックを打つ際の体重移動とバランス

4. **フットワーク**
   - フットワークの特徴（前後左右の動き）
   - ステップの仕方
   - 距離感の取り方
   - ポジショニングの特徴

5. **防御技術**
   - ガードの仕方
   - 回避の仕方（スウェイ、ダッキングなど）
   - 防御のタイミング
   - 防御からカウンターへの切り替え

6. **癖や特徴的な動き**
   - 特徴的な動きや癖
   - よく使うコンビネーション
   - 試合運びの特徴
   - 心理的な特徴（攻撃的、カウンター重視など）

7. **総合的なスタイル**
   - 攻撃的なスタイルか、カウンター重視か
   - スタンド重視か、グラウンド重視か
   - 戦術的な試合運びか、本能的な試合運びか

各項目について、具体的で詳細な記述をお願いします。
```

## プロンプトのカスタマイズ

各選手の特徴に応じて、プロンプトをカスタマイズしてください：

1. **選手名を追加**：「この動画は、[選手名]の試合動画です。」
2. **注目すべき点を追加**：「特に以下の点に注目してください：[具体的な点]」
3. **特定の技術に焦点を当てる**：特定の技術（例：パンチ、キック）に焦点を当てたプロンプトを使用

## 解析結果の活用

1. **解析結果の確認**
   - 解析結果を、実際の動画と照らし合わせて確認
   - 専門知識を活用して、結果を検証

2. **データベースへの登録**
   - 解析結果を、拡張テンプレートに従って整理
   - `Fighters_Database.md`に追加または更新

3. **継続的な改善**
   - 複数の動画を解析して、より正確な情報を得る
   - ユーザーのフィードバックに基づいて、データを改善
