# 有名格闘家データベース - Difyナレッジベース用

## このデータベースについて

このデータベースは、動画解析によって得られた詳細な技術的特徴を含む、拡張された選手データベースです。
**特に、ユーザーの動画解析結果と比較して「この動きは誰々に似ている」「この動きは誰々を目指している」と判断できるレベルの詳細なデータを含みます。**

### データベースの使い方

1. **動画解析の実行**
   - 各選手の試合動画を準備
   - 動画解析ツール（Dify、GPT-4 Visionなど）を使用して解析
   - 解析結果を取得

2. **データの整理**
   - 解析結果を、このフォーマットに従って整理
   - 専門知識を活用して、結果を補完・修正
   - **重要**：「ユーザーとの比較に使用する特徴的な動きのリスト」を必ず作成

3. **データベースへの登録**
   - 整理したデータを、このファイルに追加または更新
   - Difyのナレッジベースにアップロード

4. **ユーザーの動きとの比較**
   - ユーザーの動画解析結果を取得
   - 各選手の「ユーザーとの比較に使用する特徴的な動きのリスト」と照合
   - 類似度を計算し、やる気を伸ばすための提示を行う

5. **継続的な改善**
   - 複数の動画を解析して、より正確な情報を得る
   - ユーザーのフィードバックに基づいて、データを改善
   - ユーザーの動きとの比較精度を向上させる

### データフォーマットの説明

各選手のデータは、以下のセクションで構成されています：

- **基本情報**：階級、主な団体、主な実績、ニックネーム、身長、リーチなど
- **スタンスの詳細分析**：基本スタンス、スタンスの動き方、特徴的な動き
- **パンチの詳細分析**：ストレート、フック、アッパーカット、コンビネーションなど
- **キックの詳細分析**：フロントキック、ミドルキック、ハイキックなど（該当する場合）
- **フットワークの詳細分析**：前後の動き、左右の動き、ステップの仕方、距離感の取り方、ポジショニング
- **防御技術の詳細分析**：ガードの仕方、回避の仕方、防御のタイミング、防御からカウンターへの切り替え
- **癖や特徴的な動きの詳細分析**：特徴的な動きや癖、よく使うコンビネーション、試合運びの特徴、心理的な特徴
- **総合的なスタイルの詳細分析**：攻撃スタイル、試合運び、スタンドとグラウンド、心理的な特徴、総合的な評価
- **ユーザーとの比較に使用する特徴的な動きのリスト**：ユーザーの動きと比較するためのチェックリスト
- **参考になる技術**：ユーザーが参考にすべき技術の詳細
- **参考動画**：代表的な試合動画のURLまたは説明

#### スタンスの詳細分析

**基本スタンス**
- **スタンスの種類**：オーソドックス（右利き）
- **スタンスの幅**：肩幅の1.2倍、約60cm
- **重心の位置**：前足に60%の体重、後足に40%の体重
- **スタンスの角度**：前足45度、後足90度
- **スタンスの高さ**：膝が約30度曲がっている
- **スタンスの特徴**：
  - バランスが良く、前後左右に素早く動ける
  - スタンスの幅を状況に応じて調整する（攻撃時は約65cm、防御時は約55cm）
  - 攻撃時は前重心（前足70%、後足30%）、防御時は後重心（前足50%、後足50%）にシフト

**スタンスの動き方**
- **前進時のスタンス変化**：前足を約30cm前に出し、後足を約20cm前に出す、この動作を0.3秒で行う
- **後退時のスタンス変化**：後足を約30cm後ろに引き、前足を約20cm後ろに引く、この動作を0.3秒で行う
- **左右への動き**：左にステップする際は、左足を約40cm左に移動し、右足を約30cm左に移動、この動作を0.4秒で行う
- **前後の動きのスピード**：1秒間に約3回前後に動く
- **左右の動きのスピード**：1秒間に約2回左右に動く

**スタンスの特徴的な動き**
- **攻撃前のスタンス変化**：パンチを打つ前に、前足に体重を70%移動させる、この動作を約0.1秒で行う
- **防御前のスタンス変化**：防御する前に、後足に体重を60%移動させる、この動作を約0.1秒で行う
- **特徴的な癖**：常に左足を少し前に出す癖がある（約5cm前）

#### パンチの詳細分析

**ストレート**
- **軌道**：直線的で、わずかにアッパー気味
- **角度**：水平から約8度上向き、体の中心線から約5度右
- **スピード**：0.15秒で打ち抜く、平均的なストレートの1.3倍の速度
- **威力**：体重の1.5倍の力で打つ、平均的なストレートの1.2倍の威力
- **体重移動**：前足に体重を70%移動させ、腰を約15度回転させる
- **腕の軌道**：肩から約45度の角度で前方に伸ばす
- **手首の角度**：手首を約10度下向きに曲げる
- **特徴的な動き**：
  - ストレートを打つ前に、わずかに頭を約5cm下げる癖がある
  - ストレートを打つ際に、左肩を約10度前に出す
  - ストレートを打った後、すぐにガードに戻る（約0.1秒）
- **よく使う場面**：カウンターストレート、距離を詰めるためのストレート
- **命中率**：約85%（試合データより）

**フック**
- **軌道**：水平からややアッパー気味
- **角度**：体の中心線から約90度横、水平から約12度上向き

### データベースの継続的な改善

1. **動画解析の精度向上**
   - 複数の動画を解析して、より正確な情報を得る
   - 異なる試合での動きの一貫性を確認
   - 試合の状況（序盤、中盤、終盤）による動きの違いを分析

2. **ユーザーフィードバックの活用**
   - ユーザーからのフィードバックに基づいて、データを改善
   - ユーザーの動きとの比較精度を向上させる
   - やる気を伸ばすための提示方法を改善

3. **新しい選手の追加**
   - 新しい選手の動画を解析して、データベースに追加
   - ユーザーからの要望に応じて、特定の選手の詳細分析を追加

### データベースの活用方法

このデータベースは、AIKA18の動画解析機能で以下のように活用されます：

1. **ユーザーの動きの特徴を抽出**
   - ユーザーの動画解析結果から、技術的特徴を詳細に抽出

2. **データベースとの照合**
   - 各選手の「ユーザーとの比較に使用する特徴的な動きのリスト」と照合
   - 一致する特徴を数え、類似度を計算

3. **やる気を伸ばすための提示**
   - 類似度が高い選手を見つけたら、必ずユーザーに伝える
   - 性別に応じた適切な言葉で、やる気を伸ばす
   - 女性ユーザーには優しく褒める、男性ユーザーには厳しくも愛情を持って励ます
